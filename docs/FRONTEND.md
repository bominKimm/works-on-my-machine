# React.js 프론트엔드 구현 가이드

> **Status:** 개발 예정 (Planning)

## 개요

SecurityBlueprint Agent의 프로덕션 환경용 프론트엔드는 React로 구현될 예정입니다. 현재 `streamlit_app/`의 Streamlit UI는 로컬 데모 및 개발 테스트 목적의 임시 구현입니다.

## 프로젝트 위치

```
/
├── frontend/              # React 프로젝트 (이 디렉토리에 구현 예정)
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── ...
├── api/                   # 기존 FastAPI 백엔드 (변경 없음)
└── streamlit_app/         # Streamlit UI (로컬 데모용)
```

## 기술 스택

기술 스택:
- Framework: React
- Language: TypeScript
- State Management: Zustand
- UI Library: shadcn/ui + Tailwind CSS
- Build Tool: Vite
- Routing: React Router

## API 통신

React 프론트엔드는 기존 FastAPI 백엔드의 엔드포인트를 사용합니다:

| Method | Endpoint          | 설명                      |
| ------ | ----------------- | ------------------------- |
| GET    | `/api/v1/health`  | 헬스 체크                 |
| POST   | `/api/v1/analyze` | 아키텍처 파일 분석        |
| POST   | `/api/v1/chat`    | LLM API 호출        |

자세한 API 명세는 `/docs` (Swagger UI)를 참조하세요.

### 화면 상태별 동작

**상태 1: 초기 (Idle)**
- 업로드 카드만 표시 (중앙 정렬, 크게)
- 파이프라인 바는 모든 단계가 비활성 상태
- 배경: 그라데이션 + 애니메이션

**상태 2: 업로드 완료**
- 업로드 카드에 파일 정보 표시
- "Start Analysis" 버튼 활성화
- 부드러운 진동 애니메이션

**상태 3: 분석 진행 중**
- 업로드 카드 축소 (상단으로 이동)
- 분석 진행 카드 확장 표시
- 파이프라인 바 실시간 업데이트
- 로딩 애니메이션 (iOS 스타일)

**상태 4: 분석 완료**
- 업로드 카드 더 축소 (컴팩트 모드)
- 결과 요약 카드 표시
- 상세 결과 카드 Grid 표시
- AI Advisor FAB 활성화
- 완료 햅틱 피드백 (웹 지원 시)

### 화면 전환 흐름

```
[초기 화면]
    ↓ (파일 업로드)
[업로드 완료 - 버튼 활성화]
    ↓ (분석 시작)
[분석 진행 - 카드 확장/축소 애니메이션]
    ↓ (완료)
[결과 표시 - 카드 Grid + FAB]
    ↓ (카드 클릭)

    ↓ (FAB 클릭)
[AI 패널 슬라이드인]

[초기 화면]
```

### 컴포넌트 상세 설명

#### 1. 업로드 카드 (Glassmorphism)

**디자인:**
- 반투명 흰색 배경 (`background: rgba(255, 255, 255, 0.7)`)
- 블러 효과 (`backdrop-filter: blur(20px) saturate(180%)`)
- 부드러운 그림자 (`box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1)`)
- 1px 반투명 테두리 (`border: 1px solid rgba(255, 255, 255, 0.18)`)
- 둥근 모서리 (`border-radius: 24px`)

**기능:**
- 드래그 앤 드롭 영역 (드래그 시 테두리 강조)
- 파일 선택 버튼
- 지원 포맷 안내: PDF, PNG, JPG, JPEG
- 최대 크기: 20MB
- 업로드된 파일명 + 크기 표시
- Policy 검증 건너뛰기 체크박스 (iOS 스타일 토글)
- 분석 시작 버튼 (그라데이션, 활성화 시 펄스 효과)

**상태별 애니메이션:**
- 초기: 크게 표시 (중앙, 60% 너비)
- 파일 업로드: 스케일 업 + 바운스 효과
- 분석 시작: 상단으로 슬라이드 + 축소 (컴팩트 모드, 20% 높이)

#### 2. 분석 진행 카드

**디자인:**
- 업로드 카드와 동일한 Glassmorphism 스타일
- 중앙에 로딩 애니메이션 (iOS 스타일 스피너)
- 현재 단계 텍스트 표시

**애니메이션:**
- 페이드인 + 슬라이드 업
- 스피너: 부드러운 회전 + 그라데이션
- 완료 시: 체크마크 애니메이션 → 축소

#### 3. 결과 요약 카드

**레이아웃:**
- 5개 메트릭 카드 가로 배치 (균등 분할)
- 각 카드는 mini Glassmorphism 스타일
- 호버 시 살짝 상승 효과 (`transform: translateY(-4px)`)

**메트릭 카드 디자인:**
- 숫자: 대형 폰트, 그라데이션 텍스트
- 라벨: 작은 폰트, 반투명
- 아이콘: 상단 우측 (🔴 🔶 🔷 ⚪ ⚡)
- 심각도별 색상 악센트

#### 4. 결과 카드 Grid (2x2)

**카드 디자인:**
- Glassmorphism 스타일
- 큰 아이콘 중앙 상단
- 카드 제목 + 간단한 설명
- "View Details ›" 링크
- 호버: 배경 밝아짐 + 상승 효과
- 클릭: 전체화면 모달 오픈

**Grid 배치:**
```
┌──────────────────┬──────────────────┐
│ 🔴 취약점 목록    │ ⚡ 공격 시뮬레이션 │
│ 10 vulnerabilities│ 7 scenarios     │
│ [View Details ›] │ [View Details ›] │
└──────────────────┴──────────────────┘
┌──────────────────┬──────────────────┐
│ 🛡️ Policy 검증   │ 📊 보고서         │
│ 2 violations     │ Download report │
│ [View Details ›] │ [Download ↓]    │
└──────────────────┴──────────────────┘
```

#### 5. 상세 모달 (전체화면)

**디자인:**
- 어두운 반투명 오버레이 (`background: rgba(0, 0, 0, 0.4)`)
- 중앙 카드: 더 강한 Glassmorphism
- 상단 우측 닫기 버튼 (X)
- 콘텐츠 영역 스크롤 가능

**애니메이션:**
- 오픈: 배경 페이드인 + 카드 스케일 업 (0.9 → 1.0)
- 닫기: 반대 방향

**콘텐츠 (취약점 예시):**
- 심각도 필터 (Pill 버튼, iOS 스타일)
- 취약점 카드 목록 (아코디언 형태)
  - 헤더: 심각도 배지 + ID + 제목
  - 바디: 카테고리, 리소스, 설명, 수정방법
  - 확장/축소 애니메이션

#### 6. AI Advisor FAB + 슬라이딩 패널

**FAB (Floating Action Button):**
- 위치: 우하단 고정
- 디자인: 원형, 그라데이션 배경, 💬 아이콘
- 그림자: 강한 elevation
- 호버: 살짝 확대
- 뱃지: 새 메시지 있을 시 빨간 점

**슬라이딩 패널:**
- 우측에서 슬라이드인 (300ms ease-out)
- 너비: 30% (최소 400px)
- 배경: Glassmorphism
- 레이아웃:
  - 헤더: "AI 보안 어드바이저" + 닫기 버튼
  - 대화 영역: 스크롤 가능, 메시지 버블
  - 입력창: 하단 고정, iOS 스타일

**메시지 버블:**
- 사용자: 우측 정렬, 파란색 그라데이션
- AI: 좌측 정렬, 회색 Glassmorphism
- 타이핑 애니메이션 (AI 응답 시)
- 메시지 전송: 페이드인 + 슬라이드 업

**데스크톱 (1440px+):**
- 업로드 카드: 최대 너비 600px
- 결과 Grid: 2x2 (각 카드 약 45% 너비)
- AI 패널: 30% 너비 (최소 400px)

**태블릿 (768px - 1439px):**
- 업로드 카드: 최대 너비 500px
- 결과 Grid: 2x2 유지, 카드 간격 축소
- AI 패널: 40% 너비

**모바일 (< 768px):**
- 업로드 카드: 90% 너비
- 결과 Grid: 1열 (세로 스택)
- AI 패널: 전체화면 (하단에서 슬라이드업)
- FAB 크기 축소

### 접근성

- 키보드 네비게이션 지원
- Focus visible 스타일 (iOS 스타일 하이라이트)
- ARIA 레이블
- 색상 대비 WCAG AA 준수
- Reduced motion 설정 존중 (`prefers-reduced-motion`)

### 성능 최적화

- 카드 가상화 (많은 취약점 목록)
- 이미지 lazy loading
- CSS transform 사용 (GPU 가속)
- backdrop-filter fallback (지원 안 될 시 불투명 배경)

### 상세 결과 콘텐츠

**취약점 목록 (모달 내):**
- 심각도 필터 (Pill 버튼, iOS 스타일)
- 취약점별 아코디언 카드:
  - 헤더: 심각도 배지 + ID + 제목
  - 카테고리, 영향 리소스 (코드 스타일)
  - 설명, 수정 방법
  - 벤치마크 참조

**공격 시뮬레이션 (모달 내):**
- 시나리오별 확장 카드:
  - 심각도 + 공격명
  - MITRE ATT&CK 기법
  - 탐지 난이도, 발생 가능성
  - 전제 조건
  - 공격 체인 (단계별 목록)
  - 예상 피해

**Policy 검증 (모달 내):**
- 검증 상태: PASSED ✅ / FAILED ❌
- 위반 사항 (빨간색 카드)
- 권장 사항 (노란색 카드)

**보고서 (모달 내):**
- 다운로드 버튼 (상단)
- 마크다운 렌더링된 보고서
- 복사 버튼
